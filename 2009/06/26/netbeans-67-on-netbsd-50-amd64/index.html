Netbeans 6.7 on NetBSD 5.0 (Amd64)

Netbeans simply rocks! Download it from here! Since I’m using NetBSD I’ve downloaded the OS independent ZIP file, extracted it and found out my Java isn’t working well. I tried to install it from the pkgsrc. I tried sun-jdk15/sun-jre15 from the Amd64 ports. I got a segmentation fault. So if you got the same architecture, don’t install JDK/JRE from the package tree. If you’re able to emulate Linux code (throught Kernel configuration — check this article here), just download following files:

1) jdk-1_5_0_16-linux-i586.bin

2) jre-1_5_0_16-linux-i586.bin

Then make sure you extract the content in “/usr/pkg/java/sun-1.5/”. Then you should be able to run Netbeans without any errors.
